## 📖 系统介绍

汽车配件管理系统是一款面向汽车配件销售企业的综合管理软件，提供配件管理、库存管理、采购管理、销售管理、数据统计等功能。

### 适用用户

| 角色 | 适用人员 | 主要功能 |
|------|----------|----------|
| 管理员 | 系统管理员、店长 | 所有功能权限 |
| 仓库管理员 | 库管人员 | 配件、库存、采购管理 |
| 销售员 | 销售人员 | 销售订单管理 |

### 系统入口

访问系统地址（由管理员提供），推荐使用 Chrome、Edge、Firefox 等现代浏览器。

## 🔐 登录系统

### 登录步骤

1. 打开浏览器，访问系统地址
2. 进入登录页面
3. 输入用户名和密码
4. 点击「登录」按钮

### 默认账号

| 用户名 | 密码 | 角色 |
|--------|------|------|
| admin | 123456 | 管理员 |

### 注意事项

- 密码区分大小写
- 连续输错 5 次密码，账号将被锁定 30 分钟
- 登录成功后，Token 有效期为 8 小时
- 超时后需重新登录

## 📦 配件管理

### 功能说明

管理汽车配件的基础信息，包括配件编号、名称、规格、品牌、供应商、价格等。

### 配件列表

**查看配件列表**

1. 点击左侧菜单「配件管理」
2. 查看配件列表，支持分页显示

**搜索配件**

- 按配件编号搜索
- 按配件名称搜索
- 按分类筛选

### 新增配件

1. 点击「新增配件」按钮
2. 填写配件信息：
   - 配件编号（必填，唯一）
   - 配件名称（必填）
   - 规格型号
   - 品牌
   - 供应商
   - 单价
   - 分类
   - 单位
   - 最小库存（用于库存预警）
   - 描述
3. 点击「保存」

### 编辑配件

1. 在配件列表中找到目标配件
2. 点击「编辑」按钮
3. 修改配件信息
4. 点击「保存」

### 删除配件

1. 在配件列表中找到目标配件
2. 点击「删除」按钮
3. 确认删除

> **注意**：已有库存或关联订单的配件无法删除

### 配件状态

- **启用**：正常显示在列表中，可被业务操作使用
- **禁用**：不在列表中显示，不可被业务操作使用

## 📊 库存管理

### 功能说明

管理配件的库存数量，支持库存查询、调整、流水追踪等功能。

### 库存列表

**查看库存**

1. 点击左侧菜单「库存管理」
2. 查看各配件的当前库存

**搜索库存**

- 按关键词搜索（配件编号/名称）
- 低库存筛选：只显示低于安全库存的配件

### 库存调整

1. 点击「调整库存」按钮
2. 选择配件
3. 输入调整数量（正数增加，负数减少）
4. 填写调整原因
5. 点击「确认」

### 库存流水

**查看流水**

1. 点击左侧菜单「库存流水」
2. 查看库存变动记录

**流水类型**

| 类型 | 说明 |
|------|------|
| IN | 入库（采购入库） |
| OUT | 出库（销售出库） |
| TRANSFER | 调拨 |
| CHECK | 盘点 |

**搜索流水**

- 按配件筛选
- 按操作类型筛选
- 按时间范围筛选
- 按关联订单号筛选

### 低库存预警

当配件库存低于设定的最小库存时，系统会：
- 在首页显示预警信息
- 在库存列表中标记低库存配件
- 提醒及时补货

## 🛒 采购管理

### 功能说明

管理配件的采购流程，包括采购订单创建、审核、入库等。

### 采购订单列表

**查看订单**

1. 点击左侧菜单「采购管理」
2. 查看采购订单列表

**搜索订单**

- 按订单号搜索
- 按供应商搜索
- 按状态筛选

### 订单状态

| 状态 | 说明 | 操作 |
|------|------|------|
| PENDING | 待审核 | 管理员可审核 |
| APPROVED | 已审核 | 可入库 |
| COMPLETED | 已完成 | - |
| CANCELLED | 已取消 | - |

### 创建采购订单

1. 点击「新增采购订单」按钮
2. 填写订单信息：
   - 供应商名称
   - 供应商联系人
   - 供应商电话
   - 备注
3. 添加订单明细：
   - 选择配件
   - 输入采购数量
   - 输入单价
4. 点击「保存」

### 审核订单（管理员）

1. 找到状态为「待审核」的订单
2. 点击「审核」按钮
3. 确认订单信息
4. 点击「通过审核」

### 订单入库（仓库管理员）

1. 找到状态为「已审核」的订单
2. 点击「入库」按钮
3. 确认入库数量
4. 点击「确认入库」

> **注意**：入库后库存自动增加，并生成库存流水记录

### 删除订单

- 只有「待审核」状态的订单可以删除
- 已审核或已完成的订单无法删除

## 💰 销售管理

### 功能说明

管理销售订单流程，包括订单创建、出库、完成、退货等。

### 销售订单列表

**查看订单**

1. 点击左侧菜单「销售管理」
2. 查看销售订单列表

**搜索订单**

- 按订单号搜索
- 按客户名称搜索
- 按状态筛选

### 订单状态

| 状态 | 说明 | 操作 |
|------|------|------|
| PENDING | 待出库 | 仓库管理员可出库 |
| SHIPPED | 已出库 | 可完成或退货 |
| COMPLETED | 已完成 | - |
| RETURNED | 已退货 | - |

### 创建销售订单

1. 点击「新增销售订单」按钮
2. 填写订单信息：
   - 客户名称
   - 客户电话
   - 客户地址
   - 备注
3. 添加订单明细：
   - 选择配件
   - 输入销售数量
   - 输入单价
4. 点击「保存」

### 订单出库（仓库管理员）

1. 找到状态为「待出库」的订单
2. 点击「出库」按钮
3. 确认出库数量
4. 点击「确认出库」

> **注意**：出库后库存自动减少，并生成库存流水记录

### 订单完成

1. 找到状态为「已出库」的订单
2. 点击「完成」按钮
3. 确认订单完成

### 订单退货

1. 找到状态为「已出库」或「已完成」的订单
2. 点击「退货」按钮
3. 填写退货原因
4. 点击「确认退货」

> **注意**：退货后库存自动恢复，并生成库存流水记录

### 删除订单

- 只有「待出库」状态的订单可以删除
- 已出库或已完成的订单无法删除

## 📈 数据统计

### 功能说明

提供多维度数据统计和报表导出功能，支持业务分析和决策。

### 统计类型

| 统计类型 | 说明 |
|----------|------|
| 出入库统计 | 按时间统计入库和出库数量 |
| 销售统计 | 按时间统计销售金额和订单数 |
| 采购统计 | 按时间统计采购金额和订单数 |
| 库存周转率 | 分析配件库存周转情况 |

### 查看统计

1. 点击左侧菜单「数据统计」
2. 选择统计类型
3. 选择时间范围
4. 选择时间粒度（日/周/月）
5. 查看统计图表

### 导出报表（管理员）

1. 在统计页面选择统计类型
2. 设置筛选条件
3. 点击「导出 Excel」按钮
4. 下载 Excel 报表

## ❓ 常见问题

### 登录问题

**Q: 忘记密码怎么办？**

A: 请联系系统管理员重置密码。

**Q: 登录后提示 Token 过期？**

A: Token 有效期为 8 小时，过期后请重新登录。

### 配件管理

**Q: 为什么无法删除配件？**

A: 已有库存或关联订单的配件无法删除，请先处理关联数据。

**Q: 配件编号可以修改吗？**

A: 配件编号是唯一标识，创建后不建议修改。

### 库存管理

**Q: 库存数量不对怎么办？**

A: 可通过「库存调整」功能进行修正，并填写调整原因。

**Q: 如何查看库存变动记录？**

A: 在「库存流水」页面可以查看所有库存变动记录。

### 采购管理

**Q: 采购订单审核后如何修改？**

A: 已审核的订单无法直接修改，如需修改请联系管理员。

**Q: 入库后发现数量错误怎么办？**

A: 可通过「库存调整」功能进行修正。

### 销售管理

**Q: 销售订单出库后如何修改？**

A: 已出库的订单无法直接修改，如需修改请进行退货操作后重新创建。

**Q: 如何处理部分退货？**

A: 目前系统支持整单退货，部分退货需求请联系管理员处理。

### 系统使用

**Q: 系统支持哪些浏览器？**

A: 推荐使用 Chrome、Edge、Firefox 等现代浏览器的最新版本。

**Q: 系统数据会丢失吗？**

A: 系统数据存储在数据库中，建议定期备份数据库。
